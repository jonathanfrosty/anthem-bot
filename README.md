# anthem-bot
A simple discord music bot for playing YouTube audio.

## Built with
- [Node.js 16](https://nodejs.org/en/download/)
- [discord.js 13.2.0](https://www.npmjs.com/package/discord.js/v/13.2.0)
- [@discordjs/voice 0.6.0](https://www.npmjs.com/package/@discordjs/voice/v/0.6.0)
- [@discordjs/opus 0.6.0](https://www.npmjs.com/package/@discordjs/opus/v/0.6.0)
- [discord-ytdl-core 5.0.4](https://www.npmjs.com/package/discord-ytdl-core/v/5.0.4)
- [discord-youtube-api 0.1.0](https://www.npmjs.com/package/discord-youtube-api/v/0.1.0)
- [jsoning 0.12.22](https://www.npmjs.com/package/jsoning/v/0.12.22)
