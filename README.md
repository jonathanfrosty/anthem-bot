# anthem-bot
A simple Discord music bot for playing YouTube and Spotify audio.

## Built with
- [Node.js 18](https://nodejs.org/en/download/)
- [discord.js 13.2.0](https://www.npmjs.com/package/discord.js/v/13.2.0)
- [@discordjs/voice 0.16.0](https://www.npmjs.com/package/@discordjs/voice/v/0.16.0)
- [@discordjs/opus 0.9.0](https://www.npmjs.com/package/@discordjs/opus/v/0.9.0)
- [discord-ytdl-core 5.0.4](https://www.npmjs.com/package/discord-ytdl-core/v/5.0.4)
- [spotify-web-api-node 5.0.2](https://www.npmjs.com/package/spotify-web-api-node/v/5.0.2)
- [yt-search 2.10.4](https://www.npmjs.com/package/yt-search/v/2.10.4)
- [jsoning 0.12.22](https://www.npmjs.com/package/jsoning/v/0.12.22)
